# Ejemplos de Uso / Usage Examples

## Español

### Cómo usar los templates

1. Elige un estilo visual del repositorio
2. Reemplaza `{{ song_name }}` y `{{ artist_name }}` con tu información
3. Usa el prompt resultante en tu herramienta de IA favorita (DALL-E, Midjourney, Stable Diffusion, etc.)

### Ejemplo con neon-glowing.jinja2

**Template original:**
```
A neon-glowing CD cover for "{{ song_name }}" by {{ artist_name }}...
```

**Prompt personalizado:**
```
A neon-glowing CD cover for "Midnight Dreams" by Alex Rivera. Luminous neon lights effect with glowing edges, electric colors (hot pink, cyan, purple, electric blue). Dark background to enhance the glow, light trails, LED aesthetic. Cyberpunk neon signs inspiration, vibrant fluorescent colors, night-time urban energy. Modern digital art with intense color saturation and bloom effects. Square format album cover.
```

## English

### How to use the templates

1. Choose a visual style from the repository
2. Replace `{{ song_name }}` and `{{ artist_name }}` with your information
3. Use the resulting prompt in your favorite AI tool (DALL-E, Midjourney, Stable Diffusion, etc.)

### Example with cartoon-illustrated.jinja2

**Original template:**
```
A cartoon-style CD cover for "{{ song_name }}" by {{ artist_name }}...
```

**Customized prompt:**
```
A cartoon-style CD cover for "Summer Vibes" by The Groove Collective. Illustrated design with playful characters, bold outlines, vibrant colors. Comic book or animation aesthetic with expressive artwork. Hand-drawn quality with digital polish, fun and energetic illustrated elements. Modern illustration style with professional finish, creative character design or whimsical scenes. Square format album artwork, eye-catching and colorful.
```

## Tips / Consejos

### Español
- Ajusta el prompt agregando detalles específicos sobre colores o elementos que quieres ver
- Puedes combinar elementos de diferentes estilos
- Experimenta con diferentes herramientas de IA para obtener resultados variados
- Especifica "4K", "high resolution" o "ultra detailed" para mejor calidad

### English  
- Adjust the prompt by adding specific details about colors or elements you want to see
- You can combine elements from different styles
- Experiment with different AI tools to get varied results
- Specify "4K", "high resolution" or "ultra detailed" for better quality
